How to download the client

1. Download git version 2.10.0 (link at bottom)

2. Load git 2.10.0 onto your XP/Vista computer

3. Open git bash

4. type "cd Desktop"

5. Then type in this command:

git clone https://www.github.com/AWonkeyTortila/Internet

6. Open the folder where you downloaded it at (in most cases, "Desktop")

7. Open "client", then open "Main.html"


How to update the client (using git)

1. Go to the "OfflineWeb" folder, and open it.

2. Open git bash

3. then 'cd' your way to the folder

Example: "cd Desktop" then, "cd OfflineWeb"

4. then type "git pull"

5. Then reload the webpage
