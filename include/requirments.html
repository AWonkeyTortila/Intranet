System Requirements


Windows Version   Internet Explorer Version   Hard Drive Space  
XP or above Internet Explorer 7 or better 50MB

Internet Connection

An internet connection is HEAVILY reccomeded.


Minimum Screen Resolution

The minimum screen resolution for this project is 800x600, also happening to be the minimum screen resolution of WindowsXP.


If you have no acces to IE7+, or litteraly any other web browser, there is a .CSS file in .root/client/Main.css, were it has code compatible with IE6.

Please note however that this is not reccomeded, because of the fact that most developers probably wont target IE6.
