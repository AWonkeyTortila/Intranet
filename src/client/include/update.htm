How to update the client

If you have an internet conection

1. go to Intranet/src/pages/documents

2. In there, run the git install (This works on XP and above, server not tested)

3. Run git, and go to your desired folder location

4. run "git clone https://github.com/AWonkeyTortila/Intranet" if you don't have the repo

5. run "git pull" to update an existing repo

If you don't have an internet conection

1. go to https://github.com/AWonkeyTortila/Intranet

2. Download the Zip file and unzip it in your desired directory

3. go to Intranet/src/client/main.html
